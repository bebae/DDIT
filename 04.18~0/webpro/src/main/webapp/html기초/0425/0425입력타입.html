정규식 : 입력데이터의 형식을 규정
        [0-9]{3}-[0-9]{4}=[0-9]{4}
        + : 1번 이상 와야한다
        * : 0번 이상 와야한다
        ? : 0 OR 1
        () : 그룹
        . : \n를 제외한 모든 문자를 의미
        \. : "." 자체를 의미

        이메일 예시
        kk123@naver.com
        Aabc99@ddit.or.kr
        kk123@w3.org
        [a-zA-z]+[0-9]*@[a-zA-Z]+[0-9]*(\.[a-zA-Z]+){1,2}
